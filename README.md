# 📱 Pro Bulk SMS Sender

A powerful Android application for sending bulk SMS to multiple recipients with full remote control.

---

## ✨ Features

- **Bulk SMS Sending** — Send SMS to hundreds of contacts at once
- **Multi-SIM / eSIM Support** — Auto detects all active SIMs including eSIM
- **SIM Rotation** — Automatically rotates between SIM 1 and SIM 2 to avoid carrier blocking
- **Configurable Delay** — Set delay between each SMS to prevent carrier blocking
- **Import Contacts** — Load phone numbers from a text file or paste directly
- **Long SMS Support** — Automatically splits messages over 140 characters
- **Resume Sending** — Resumes from where it stopped if interrupted
- **Cancel Operation** — Cancel ongoing bulk SMS at any time
- **Default SMS App Role** — Properly requests Default SMS role on Android 10+
- **Material Design UI** — Clean and easy to use interface
